# Task Tracker

A full-stack Task Management application built using React, Node.js, Express, and MongoDB.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Complete
- Persistent Storage with MongoDB Atlas

## Tech Stack

### Frontend
- React
- Axios
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas
- Mongoose
